# 🤖 AI-Powered Test Automation Framework

A production-grade SDET portfolio project combining
Selenium, BDD, API Testing, Visual AI, and Claude AI
for automated test generation.

## 🛠️ Tech Stack
- Selenium WebDriver 4.x + Java 11
- Cucumber BDD + TestNG
- RestAssured API Testing
- Applitools Visual AI Testing
- Claude AI — auto-generates test cases from user stories
- GitHub Actions CI/CD
- ExtentReports HTML Dashboard

## 📁 Project Structure
- `pages/` — Page Object Model classes
- `tests/` — TestNG test classes
- `stepdefs/` — Cucumber step definitions
- `utils/` — BaseTest, AI Generator, helpers

## 🚀 How to Run
```bash
mvn test
```

## 🤖 AI Test Generator
Paste any user story → Claude API generates
ready-to-run Selenium tests automatically.

## 📊 Reports
After run: open `target/ExtentReport.html`
